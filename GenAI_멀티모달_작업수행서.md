# GenAI 기초 2 — 멀티모달 콘텐츠 제작 작업수행서

> 본 문서는 "GenAI 기초 2: 멀티모달 콘텐츠 제작" 과제 원문을 분석하여, 실제로 무엇을 어떤 순서로 해야 하는지 실행 가능한 단계로 재구성한 작업수행서입니다.

---

## 0. 과제 핵심 분석 요약

| 항목 | 내용 |
|---|---|
| 최종 산출물 | ① 스토리보드(기획) 문서 PDF 1개, ② 광고 영상 MP4 1개(10초 이내) |
| 평가 포인트 | 도구 사용 개수가 아니라 **기획 의도 → 프롬프트 → 결과물 → 통합 편집**으로 이어지는 파이프라인 설계와 그 근거 설명 |
| 필수 도구 종류 | 이미지 생성 AI 1종↑, 비디오 생성/변환 AI 1종↑, 오디오 생성 AI 1종↑ |
| 필수 서사 조건 | 기승전결 구조 **또는** 명확한 메시지 전달 구조 중 하나, 마지막 3~5초에 브랜드 인지 장치(로고/브랜드명/슬로건/CTA) |
| 제약 조건 | 직접 촬영·유료 스톡 영상 금지, 딥페이크·유해 콘텐츠 금지, 편집 툴은 컷 편집/자막/색보정 등 "통합 편집" 용도로만 제한 사용 |
| 리스크 관리 | 크레딧 소모 큰 비디오 생성은 이미지 확정 후 진행, 도구 대안(대체 도구) 최소 1개씩 준비 |

이 과제는 **"AI 도구 다루는 실습"이 아니라 "제작 파이프라인 설계 + 의사결정 근거 문서화"**가 본질입니다. 따라서 작업 순서도 "기획 → 문서화 → 생성 → 검증 → 편집 → 제출" 흐름으로 짜야 합니다.

---

## STEP 1. 브랜드 & 캠페인 기획 정의 (30분~1시간)

**할 일**
1. 가상 또는 실제 브랜드 1개를 선정한다. (예: 패션, 음료, 앱 서비스 등 — 비주얼/오디오로 표현하기 좋은 소재 추천)
2. 다음 항목을 문장으로 정리한다.
   - 브랜드명
   - 타겟 고객 (연령/성향)
   - 톤앤매너 (분위기, 색감, 스타일 키워드 3~5개)
   - USP(차별점): 한 문장
   - 광고 목적: 인지 / 전환 / 리텐션 중 선택
   - 핵심 메시지: 반드시 **한 문장**으로 압축

**산출물**: `브랜드 아이덴티티` 섹션 초안 (스토리보드 문서의 1페이지가 됨)

**팁**: 이 단계에서 톤앤매너를 구체적인 키워드(예: "사이버펑크, 네온, 우천의 서울")로 정해두면, 이후 모든 프롬프트에 이 키워드를 재사용해 화풍 일관성을 확보할 수 있습니다.

---

## STEP 2. 스토리 구조 및 씬 설계 (1시간)

**할 일**
1. 10초 이내 영상을 몇 개 씬으로 나눌지 결정한다. (보통 2~4씬, 씬당 2~5초 권장)
2. 다음 중 하나의 구조를 선택해 씬 흐름을 짠다.
   - **기승전결형**: 문제 제시 → 전환 → 해결/제안
   - **메시지 전달형**: "누구에게 어떤 가치를 주는가"가 드러나는 구조
3. **마지막 3~5초는 반드시** 로고/브랜드명/슬로건/CTA 중 하나가 등장하도록 설계한다.
4. 각 씬마다 아래 필수 필드를 채울 표를 준비한다 (스토리보드 문서 양식):
   - 씬 번호 / 씬 길이(초)
   - 목표 메시지(한 문장)
   - 화면 구성(구도/피사체/배경/텍스트 유무)
   - 내레이션 또는 화면 카피
   - 사용 도구(이미지/비디오/오디오)와 사용 목적
   - 입력 프롬프트(원문) + 출력 결과 요약(한 줄)
   - 생성 결과 파일명

**산출물**: 씬 리스트 표 (빈 칸 상태, STEP 4~6에서 채워짐)

---

## STEP 3. 생성 도구 선정 및 대체 도구 확보 (30분)

**할 일**
1. 이미지 / 비디오 / 오디오 각 영역에서 **주 도구 1개 + 대체 도구 1개 이상**을 정한다.
   - 이미지: 예) Midjourney (주) / DALL·E, Stable Diffusion (대체)
   - 비디오: 예) Runway (주) / Pika, Kling (대체)
   - 오디오: 예) Suno (주) / ElevenLabs, Udio (대체)
2. 각 도구를 "왜 그 목적에 선택했는지" 한두 줄로 메모한다. (스토리보드에 그대로 기입)
3. 캐릭터/스타일 일관성이 필요하면 각 도구의 Character Reference(cref) / Style Reference(sref) / ControlNet 기능 사용 여부를 결정하고, 사용한다면 어떤 파라미터를 고정할지 미리 적어둔다.

**산출물**: 도구 선정표 (주 도구/대체 도구/선택 이유/일관성 유지 방법)

---

## STEP 4. 이미지 생성 (키 비주얼 확정) — 크레딧 관리의 핵심 단계

**할 일**
1. 씬별로 프롬프트를 작성해 이미지를 생성한다. 톤앤매너 키워드를 모든 프롬프트에 공통으로 포함시켜 스타일을 통일한다.
2. 결과가 마음에 들지 않으면 프롬프트를 수정하며 재생성한다. **이 중 최소 1개 씬은 "수정 전 프롬프트 → 문제점 → 수정 후 프롬프트 → 결과 변화"를 반드시 기록**한다. (평가 필수 항목)
3. 모든 씬의 키 비주얼(정지 이미지)이 확정될 때까지 이 단계를 반복한다. **영상 변환은 이미지가 확정된 후에 시작**한다 (크레딧 절약 전략).
4. 파일명 규칙을 정해 저장한다. (예: `scene01_keyvisual.png`)

**산출물**: 씬별 확정 이미지 파일 + 프롬프트 수정 전/후 기록 1건 이상

---

## STEP 5. 비디오 변환 & 오디오 생성

**할 일**
1. 확정된 이미지를 비디오 생성/변환 도구에 넣어 짧은 모션(패닝/줌/루프 등)을 만든다.
2. 대사가 있는 씬이 있다면(보너스: 립싱크) 음성 합성 도구로 내레이션을 만들고 입모양 싱크를 맞춘다.
3. 배경음악/효과음을 오디오 생성 AI로 만든다. 브랜드 톤앤매너와 맞는 무드(예: 로파이, 신스웨이브 등)를 지정한다.
4. 각 씬의 결과 파일명을 스토리보드에 기입한다.

**크레딧 부족 시 대응 (택 1 이상)**
- 씬 수 축소 (예: 6씬 → 4씬)
- 정지 이미지 + 짧은 모션 중심으로 구성
- 스타일 레퍼런스 고정으로 재생성 횟수 줄이기

**산출물**: 씬별 비디오 클립 + 오디오 파일

---

## STEP 6. 통합 편집 (컷 편집 단계)

**할 일**
1. 편집 툴(Premiere Pro, CapCut 등)에서 각 씬 클립을 순서대로 배치한다.
2. 자막, 간단한 색보정, 오디오 레벨 조정만 진행한다. **핵심 비주얼/오디오를 편집 툴에서 새로 만들지 않도록 주의** (제약 조건 위반).
3. 전체 길이가 10초를 넘지 않는지 확인한다.
4. 최종 인코딩: 해상도 1080p(최소 720p), 24~30fps, H.264/AAC 권장.
5. 파일명 규칙에 따라 최종 파일을 저장한다. (예: `브랜드명_branding_video_팀명.mp4`)

**산출물**: 최종 MP4 파일 (10초 이내)

---

## STEP 7. 스토리보드 문서 최종 정리 (PDF화)

**할 일**
1. 지금까지 채운 내용을 다음 순서로 정리한다.
   1) 브랜드 아이덴티티
   2) 캠페인 목표 / 핵심 메시지
   3) 씬 구성 표 (필수 필드 전부 포함)
   4) 사용 도구 목록 (주/대체, 선택 이유)
   5) 프롬프트 수정 전/후 기록 (최소 1개 씬)
   6) 최종 영상 파일 정보 (파일명/길이/해상도/코덱)
2. 누락된 필드가 없는지 아래 체크리스트로 검증한다.
3. Word 또는 문서 도구에서 작성 후 PDF로 내보낸다.

**산출물**: 스토리보드 PDF 1개

---

## STEP 8. 제출 전 최종 체크리스트

- [ ] 브랜드 아이덴티티(타겟/톤앤매너/USP) 정의됨
- [ ] 핵심 메시지 한 문장으로 명확함
- [ ] 모든 씬에 필수 필드(번호/길이/메시지/화면구성/내레이션/도구/프롬프트/결과요약/파일명) 기입됨
- [ ] 최소 1개 씬 프롬프트 수정 전/후 기록 있음
- [ ] 이미지·비디오·오디오 생성 도구 각 1종 이상 사용 및 선정 이유 기록
- [ ] 영상 길이 10초 이내
- [ ] 기승전결 또는 명확한 메시지 전달 구조 중 하나 충족
- [ ] 마지막 3~5초에 브랜드 인지 장치(로고/브랜드명/슬로건/CTA) 포함
- [ ] 직접 촬영/유료 스톡 영상 미사용, 딥페이크·유해 콘텐츠 없음
- [ ] 편집 툴은 컷 편집/자막/색보정/오디오 레벨 조정 용도로만 사용됨
- [ ] 스토리보드 PDF + 광고 영상 MP4 준비 완료

---

## (선택) STEP 9. 보너스 과제

시간이 남는다면 아래 중 택1 이상 도전:

1. **립싱크**: 인물 발화 장면 1개 이상 추가, 입모양-대사 자연스럽게 매칭
2. **동일 스토리보드 다른 도구 재제작**: 이미지 또는 비디오 파트를 다른 도구로 바꿔 1~2개 씬 재제작 → 결과 비교 기록
3. **화면 비율 버전**: 16:9(유튜브) + 9:16(숏폼) + 1:1(피드) 중 2개 이상 제작

## 전체 작업 순서 한눈에 보기

```
STEP 1 브랜드/캠페인 기획
   ↓
STEP 2 씬 구조 설계 (기승전결/메시지 전달 + 마지막 3~5초 브랜드 노출)
   ↓
STEP 3 도구 선정 (주+대체, 선택 이유 기록)
   ↓
STEP 4 이미지 생성 확정 (프롬프트 수정 전/후 최소 1건 기록) ★크레딧 절약 핵심
   ↓
STEP 5 비디오 변환 + 오디오 생성
   ↓
STEP 6 통합 편집 (컷/자막/색보정만)
   ↓
STEP 7 스토리보드 문서 정리 → PDF 변환
   ↓
STEP 8 최종 체크리스트 검증 → 제출
   ↓
(선택) STEP 9 보너스 과제
```

이 순서를 따르면 "도구를 몇 개 썼는가"가 아니라 "왜 이 도구, 이 프롬프트, 이 구조를 선택했는가"를 설명할 수 있는 상태로 과제를 완성하게 됩니다.
